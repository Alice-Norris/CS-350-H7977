 ### CS-350 Thermostat Project
This project had several requirements:
Two buttons control the set temperature of the thermostat, checking if buttons have been pressed every 200ms to raise or lower set temperature.
The temperature sensor checks the environmental temperature every 500ms, power on an LED when the simulated heater is on (the environmental temperature is less than the set temperature) or turning it off when the heater is off.
